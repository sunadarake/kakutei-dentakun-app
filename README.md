# 確定申告足し算くん

確定申告とかで使える足し算アプリ。

半角、全角の数字に対応しており、途中に , を入れても機能する。

## 使い方

テキストボックスに数字を入れていく。
それぞれの数字は　半角・全角スペースやタブ、改行で区切ること。

## 環境

- node.js v22.17.0

## Install

```
git clone git@github.com:sunadarake/kakutei-dentakun-app.git
kakutei-dentakun-app
npm install

# web で使う
npm run dev

# デスクトップアプリを作る
npm run build
```
